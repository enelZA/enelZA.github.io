Use Java PlantUMl.jar to create sequence and activity diagrams

2014-06-03 by EnelZA, tagged as howto, java

Download PlantUML from sourceforge.net

Add the following to a text file

@startuml
|Swimlane1|
start
:foo1;
|#AntiqueWhite|Swimlane2|
:foo2;
:foo3;
|Swimlane1|
:foo4;
|Swimlane2|
:foo5;
stop
@enduml

Use the plantuml.jar file to generate the swimlane activity diagram.

java -jar plantuml.jar file.txt
