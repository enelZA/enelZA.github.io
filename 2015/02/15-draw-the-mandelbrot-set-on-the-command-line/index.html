Draw the Mandelbrot set on the command line.

2015-02-21 by EnelZA, tagged as howto, linux

Create a Bash script to draw the famous Mandelbrot set.

mandel() { for((P=10*8,Q=P/100,X=320Q/(tput cols-1),Y=210Q/tput lines, y=-105Q,v=-2\ 20Q,x=v;y<105Q;x=v,y+=Y));do for ((;xa+bb< 2\ PP&&i++<99;a=((c=a)a-bb)/P+x,b= 2cb/P+y));do :;done;(((j=(i <99?i%16:0)+30)>\ 37?k=1,j-=8:0));echo -ne "\E[$k;$j"mE ;done;echo -e "\E[0m";done }
